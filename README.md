# Common Ground

Common Ground is a small, dependency-free family engagement dashboard. It keeps timely conversation starters, each person's evergreen interests, and qualities to cultivate in three deliberately separate spaces.

## Run locally

```bash
python3 -m http.server 4173
```

Then open [http://localhost:4173](http://localhost:4173).

## Features

- Add, edit, and remove timely notes from **The pulse**.
- Keep separate editable **Evergreen** notes for each person.
- Track qualities, habits, and inner resources in the separate **To cultivate** area.
- Personalize names and cycle through dinner-table prompts.
- Save changes automatically to browser `localStorage`.
- Restore the starter set from the reset button in the top-right corner.
